# 🎮 三角洲护航 SaaS 系统

专业游戏代练护航服务平台，支持用户端H5、接单员端、PC管理后台三端。

## 功能特性

- **用户端**：注册登录、浏览商品、下单支付、订单管理、充值、申请接单员
- **接单员端**：抢单大厅、进度更新、完成订单、佣金结算、提现
- **管理后台**：仪表盘、订单派单/退款、接单员审核、商品/分类/轮播图/公告管理、提现审核、系统配置

## 技术栈

- **后端**：Express + SQLite (better-sqlite3) + JWT + bcryptjs
- **前端**：原生 HTML/CSS/JS，移动端 H5 风格
- **数据库**：SQLite，12张表

## 快速开始

```bash
npm install
node server.js
```

- 用户端：http://localhost:3001
- 管理后台：http://localhost:3001/admin
- 接单员端：http://localhost:3001/player
